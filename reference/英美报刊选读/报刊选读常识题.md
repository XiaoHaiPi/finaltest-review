报刊选读常识题

|       Element       |                          Definition                          |
| :-----------------: | :----------------------------------------------------------: |
|      Headline       |           Catches your attention/Sums up the story           |
|       Byline        | Writer's name/Writer's Specialty, e.g. sprots,food,crime,current events |
| Placeline(optional) |                    Where the story begins                    |
|        Lead         | The opening section/Gives most important information/Should answer most of the 5W's |
|        Body         |      Supplies detail/Most important details come first       |
|        Facts        |                    Simple true statements                    |
|      Quotation      | What someone actually said/Adds accuracy/Adds "at the scene" feeling |

|       元素        |                     定义                      |
| :---------------: | :-------------------------------------------: |
|       标题        |            吸引你的注意力/总结故事            |
|      署名行       | 作家姓名/作家专业，例如体育、食品、犯罪、时事 |
| Placeline（可选） |                故事开始的位置                 |
|       引导        |   开头部分/提供最重要的信息/应回答大部分5W    |
|       正文        |           提供细节/最重要的细节优先           |
|       事实        |                简单真实的陈述                 |
|       引文        |    某人实际说的话/增加准确性/增加“现场”感     |
